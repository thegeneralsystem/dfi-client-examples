# DFI-API-Examples

A set of example apps for interacting with the DFI API

## Installation

```sh
npm install
```

## Example App: Taxi2Dfi

### Usage

```sh
npm run taxi2dfi -- --help
```

e.g.

```sh
npm run taxi2dfi -- ~/yellow_tripdata_2009-04.parquet -t MY_API_TOKEN
```

More info: [README](apps/taxi2dfi/README.md)

## Example App: Bus2Dfi

### Usage

```sh
npm run bus2dfi -- --help
```

e.g.

```sh
npm run bus2dfi -- -t MY_API_TOKEN
```

More info: [README](apps/bus2dfi/README.md)
